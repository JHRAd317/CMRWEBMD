Install the Smart Agent

  1. Download the following three files as .txt using the buttons below.
    (Files are provided as .txt to work with all browsers and security settings.)
  2. After downloading:
    Rename activate.txt to activate.ps1 on your computer.
    (Right-click the file, select Rename, and change activate.txt to activate.ps1.)
  3. Run activate.ps1 with PowerShell:
    Right-click the file and select “Run with PowerShell”.
  4. That’s it! The installer will finish setup and automatically run your agent.

Why .txt? For security, browsers don’t allow direct downloading of .ps1 files. By downloading as .txt and renaming locally, you get a safe, working install every time.
