Smart Agent Bootstrap TEST

  1. Click each button to download Structure.txt, Listener.txt, and activate.ps1.
  2. Run activate.ps1 in PowerShell (right-click, "Run with PowerShell" or execute manually).
  3. The installer will create the structure, set up your listener, and start a local agent server (on port 5123).
  4. In your browser, visit http://localhost:5123/ - you should see "Agent is alive!" if the server is running.
Note: Make sure Structure.txt, Listener.txt, and activate.txt are hosted alongside this dashboard file on your web server for this test to function.
The Python agent requires Python to be installed and available in the system PATH.
