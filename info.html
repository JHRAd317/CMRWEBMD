About MiniDash Web

CMR MiniDash Web is a browser‑based dashboard designed to replace desktop automation tools with a clean, mobile‑friendly interface. It provides quick access to CPT/ICD lookups, customizable buttons, and workflow shortcuts.

Features

  • 4×3 main navigation grid
  • Live CPT and ICD search tools
  • Customizable button editor with local storage
  • Mobile‑friendly layout for iPhone, iPad, and desktop
  • Lightweight, fast, and fully client‑side

Version

MiniDash Web Version 1.0
Last updated: March 2026

Developer Notes

This project is designed for modular expansion. Additional pages, tools, and integrations can be added without changing the core layout. The Utility page supports button customization using browser localStorage.

Support

For updates, enhancements, or troubleshooting, return to the Utility page or contact your system administrator.
